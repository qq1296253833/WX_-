# WX_erlongMovies
微信小程序（二龙电影）
自主开发的一款可以查询电影信息的微信小程序，主要实现功能：搜索，跳转，展示电影详情信息等
目前数据是静态的数据，还未调用接口去实现网络请求
审核已经通过，目前有可用的测试版本，搜索二龙电影即可查询
